# Hook Template

Use this template to generate or review ad hooks.

Main site: https://aiads.im

## Hook brief

```yaml
product:
category:
audience:
pain_point:
desired_outcome:
platform:
offer:
tone:
proof_point:
common_objection:
```

## Hook formula library

### Pain point hook

```text
Still dealing with {{pain_point}}?
```

### Curiosity hook

```text
I did not expect {{product}} to fix this.
```

### Mistake hook

```text
Stop doing {{old_method}} before you try this.
```

### Comparison hook

```text
I compared {{product}} against {{alternative}}.
```

### Demo hook

```text
Watch what happens when I use {{product}} for {{use_case}}.
```

### Objection hook

```text
I thought {{objection}}, but then I tried this.
```

## Output checklist

A strong hook should be:

- short
- visual
- specific
- platform-native
- connected to a real product benefit
- safe from fake claims
